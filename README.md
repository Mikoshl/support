# Сайт поддержки для Андрея

Спокойное приглашение без автоматической отправки данных и без внешних сервисов.

## Публикация

1. Создайте новый публичный репозиторий GitHub.
2. Загрузите в корень `index.html`, `styles.css` и `script.js`.
3. Откройте **Settings → Pages**.
4. Выберите **Deploy from a branch**, ветку `main`, папку `/ (root)`.
5. Нажмите **Save**.

Сайт не собирает и никуда не отправляет данные. На телефоне кнопка ответа вызывает системное меню «Поделиться», а на компьютере копирует готовое сообщение.
